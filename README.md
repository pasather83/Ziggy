# Ziggy Staff Assistant

Ziggy is a Hotel Indigo Dubai staff assistant for guest/product knowledge, SOP guidance, shift handovers, service recovery, and quick operational notes.

## Deployment

This repo is connected to Vercel:

- Vercel project: `philips-projects-ab6c1a0a/ziggy`
- GitHub repo: `pasather83/Ziggy`

Every push to `main` should automatically trigger a new Vercel production deployment.

## Update Flow

```powershell
git add .
git commit -m "Update Ziggy"
git push
```

Vercel will build and deploy from GitHub after the push.
